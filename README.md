# Calculator App

Flutter-приложение калькулятора для Android.

## Возможности

- Базовые арифметические операции: `+`, `-`, `×`, `÷`
- Функции: `AC` (сброс), `±` (смена знака), `%` (процент)
- Тёмная тема в стиле iOS Calculator
- Валидация деления на ноль

## Запуск

```bash
flutter pub get
flutter run
```

## Требования

- Flutter SDK >= 3.0.0
- Android SDK
